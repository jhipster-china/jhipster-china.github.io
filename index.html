JHipster 是什么？

+
=

JHipster 是一个创建、开发、部署 Spring Boot + Angular 架构的 Web 应用或 Spring 微服务应用的开发平台。

目标

我们的目标是帮助你生成复杂的 Web 应用或微服务应用架构，来统一：

  • 高可用、健壮的 JAVA 技术栈服务端 Spring Boot 程序
  • 一个顺滑的、现代的、移动优先的前端 Angular 及 Bootstrap 程序架构
  • 一个健壮的微服务架构包括：JHipster Registry, Netflix OSS, ELK stack 以及 Docker
  • 强大的应用构建流程，使用：Yeoman, Webpack/Gulp 以及 Maven/Gradle

JHipster 快速上手

安装 JHipster yarn global add generator-jhipster

创建目录，进入 mkdir myApp && cd myApp

执行 JHipster，根据提示设置应用 jhipster

使用 JDL Studio 编辑实体对象，下载 jhipster-jdl.jh 文件

使用命令 jhipster import-jdl jhipster-jdl.jh 来创建实体对象及相关类

* 假定你已经安装了 Java, Git, Node.js, Yarn 和 Yeoman. 如果要使用 AngularJS 1, 你还需要安装 Bower 和 Gulp
